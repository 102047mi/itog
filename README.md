## Описание

Это простейшая система блога на PHP с возможностью добавления, просмотра и управления постами.

## Установка

1. Скопируйте папку simple_blog в директорию вашего веб-сервера 
2. Создайте базу данных blog и выполните SQL-запрос для создания таблицы posts.
3. Запустите веб-сервер и MySQL. 
4. Для доступа к админ-панели используйте логин admin и пароль 123.

## Функциональность

- Просмотр списка постов.
- Чтение полного текста поста.
- Добавление новых постов через админ-панель.
- Удаление постов из админ-панели.
